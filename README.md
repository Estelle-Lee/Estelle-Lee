# 이보경 (Bokyung Estelle Lee)

**백엔드 엔지니어**

복잡한 비즈니스 로직을 안정적인 시스템으로 구현하고, 데이터 기반으로 성능 병목을 해결하는 백엔드 엔지니어입니다. 차세대 학습 플랫폼(LXP) 구축, 이기종 DBMS 마이그레이션, FFMPEG 미디어 프로세싱, 시스템 자동화 등 기술 난도가 높은 영역을 실무로 다뤄왔습니다.

- **성능 최적화** — APM 기반 병목 진단으로 쿼리 수행 시간 6.0s → 0.9s (약 85% 단축), 부하율 0% 달성
- **이기종 마이그레이션** — Oracle → Tibero 무중단 데이터 이관 및 쿼리 재설계
- **미디어 자동화** — FFMPEG 실시간 스트리밍 최적화 및 코덱·키프레임 자동 관리
- 영국 글래스고 대학 소프트웨어 개발 석사 · 글로벌 협업 역량
